
# ফেসবুক রিয়েকশন এক্সচেঞ্জ অ্যাপ - ভেরসেল সেটআপ গাইড

এই প্রকল্পটি ভেরসেল (Vercel) এ ডেপ্লয় করার জন্য সম্পূর্ণ অপ্টিমাইজড। নিচে ধাপে ধাপে নির্দেশাবলী দেওয়া হলো:

### ১. গিটহাব রিপোজিটরি তৈরি (GitHub Setup)
- আপনার গিটহাব অ্যাকাউন্টে একটি নতুন রিপোজিটরি তৈরি করুন।
- এই ফাইলের কোডগুলো সেখানে পুশ (Push) করুন।

### ২. মঙ্গোডিবি সেটআপ (MongoDB Setup)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) এ একটি ফ্রি ক্লাস্টার তৈরি করুন।
- `Database Access` থেকে একটি ইউজার তৈরি করুন।
- `Network Access` থেকে IP Address `0.0.0.0/0` (Allow all) করে দিন।
- কানেকশন স্ট্রিংটি কপি করুন (যেমন: `mongodb+srv://...`)।

### ৩. ভেরসেল ডেপ্লয়মেন্ট (Vercel Deployment)
- [Vercel](https://vercel.com/) এ লগইন করুন।
- `Add New Project` এ গিয়ে আপনার গিটহাব রিপোজিটরি সিলেক্ট করুন।
- `Environment Variables` সেকশনে নিচের ভেরিয়েবলগুলো যোগ করুন:
    - `MONGODB_URI`: আপনার মঙ্গোডিবি স্ট্রিং।
    - `FB_APP_ID`: আপনার ফেসবুক অ্যাপ আইডি।
    - `FB_APP_SECRET`: আপনার ফেসবুক অ্যাপ সিক্রেট।

### ৪. ফেসবুক অ্যাপ সেটআপ (Facebook App)
- [Meta for Developers](https://developers.facebook.com/) এ গিয়ে একটি অ্যাপ তৈরি করুন।
- `Facebook Login` সেটআপ করুন।
- Valid OAuth Redirect URIs এ আপনার ভেরসেল অ্যাপের ইউআরএল যোগ করুন।

### কোড স্ট্রাকচার:
- `/api/auth/login`: ইউজারের ফেসবুক টোকেন ভেরিফাই এবং ডাটাবেস এ ইউজার সেভ করার জন্য।
- `/api/exchange/earn`: কয়েন ভেরিফিকেশন লজিক।
- `/lib/db.ts`: ডাটাবেস কানেকশন হ্যান্ডেলার।

**দ্রষ্টব্য:** এই অ্যাপটি মোবাইল-ফার্স্ট ডিজাইনে তৈরি করা হয়েছে যাতে ইউজাররা ফোন থেকে সহজেই ব্যবহার করতে পারে।
